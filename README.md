# UnityInFlow Homebrew Tap

Homebrew formulae for the [UnityInFlow](https://github.com/UnityInFlow) tools.

```bash
brew tap unityinflow/tap
brew install unityinflow/tap/<formula>
```

## Formulae

- **prompt-vc** — version control for system prompts: semantic diff, A/B comparison runs, changelogs. → https://github.com/UnityInFlow/prompt-vc
- **agent-memory** — persistent agent memory. → https://github.com/UnityInFlow/agent-memory
